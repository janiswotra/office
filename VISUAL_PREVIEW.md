## Responsive Behavior

### Desktop (1920px+)
- 3-column layout (2 cols agents, 1 col activity)
- Agent cards 2x3 grid
- Activity feed sticky sidebar

### Laptop (1280px)
- Same 3-column layout, tighter spacing
- Agent cards slightly smaller

### Tablet (768px)
- 2-column layout
- Activity feed below agents
- Agent cards 2x3 grid

### Mobile (375px)
- Single column
- Stats bar: 2x2 grid
- Agent cards: 1 column stack
- Activity feed: Horizontal scroll or collapsed

## Micro-interactions

1. **Hover Effects**
   - Cards scale up 5%
   - Border brightens
   - Smooth 300ms transition

2. **Status Changes**
   - Fade transition between states
   - Pulsing starts/stops smoothly

3. **Activity Feed**
   - New items slide in from right
   - Old items fade out at bottom
   - Staggered animation (50ms delays)

4. **Loading State**
   - Large pulsing dragon emoji
   - "Initializing Dragon HQ..." text
   - Fade out when loaded
